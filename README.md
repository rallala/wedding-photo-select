# 웨딩 사진 셀렉 📸

신랑·신부가 각자 폰/노트북으로 **실시간 동시에** 사진을 고르고, 확정한 셀렉본을 새 폴더로 복사해 작가에게 바로 전달하는 로컬 웹앱입니다.

- ⚡ **더블 클릭 실행** — 터미널 명령어 입력 필요 없이 파일만 누르면 브라우저 자동 오픈
- 📱 **스마트폰 QR 코드 접속** — 화면 상단 [📱 폰 접속 / QR] 버튼으로 카메라만 대면 즉시 접속
- 🌐 **외부 접속 지원** — 같은 와이파이가 아닌 외부(LTE/5G/다른 장소)에서도 신부/신랑이 폰으로 접속 가능
- 🐳 **Docker 지원** — Node.js 설치 없이 Docker로도 즉시 구동 가능 (`node:24-alpine`)
- 🔒 **PIN 잠금** — 접속 시 4자리 PIN 필요(상대방에겐 PIN과 QR/주소만 알려주면 됨)
- 📁 **폴더를 브라우저에서 선택** — 실행할 때 경로를 붙일 필요 없음
- 하위폴더까지 전부 펼쳐서 모든 jpg를 한 화면에
- 신랑=파랑 / 신부=분홍 / **둘 다 고른 사진(중복)=금색 테두리**
- 검색·폴더·뷰(중복만/신랑만/신부만) 필터
- 선택 내역 자동 저장(중간에 꺼져도 안 날아감)
- 확인 후 원하는 것만 새 폴더로 복사

---

## 🚀 쉬운 실행 방법 (Windows)

### 1. 같은 와이파이 / 집에서 사용할 때
`실행하기.bat` 파일을 **더블 클릭**합니다.
- 서버가 켜지면서 내 컴퓨터 웹 브라우저가 자동으로 열립니다.
- 신부/신랑님 폰은 웹 화면 상단의 **[📱 폰 접속 / QR]** 버튼을 눌러 나온 **Wi-Fi QR 코드**를 카메라로 스캔하세요.

### 2. 외부 (LTE/5G/다른 장소)에서 신부가 접속할 때
`실행_외부접속 포함.bat` 파일을 **더블 클릭**합니다.
- 외부 접속용 HTTPS 주소가 자동으로 생성됩니다.
- 웹 화면 상단의 **[📱 폰 접속 / QR]** → **[외부 접속]** 탭에 나타난 QR 코드를 폰으로 찍거나 [주소 복사]를 눌러 카톡으로 전달해 주세요.

### 3. Docker로 실행할 때
`실행_Docker.bat` 파일을 **더블 클릭**하거나 터미널에서 실행합니다:
```bash
docker run --rm -it -p 3000:3000 -v "%cd%":/app -w /app node:24-alpine node server.js
```

## 💻 터미널 실행 방식 (Mac / Linux / 개발자)

1. 터미널을 엽니다.
2. 로컬 실행:
   ```bash
   node server.js
   ```
3. 외부 접속 포함 실행:
   ```bash
   npm run tunnel
   # 또는: TUNNEL=true node server.js
   ```

---

## 💡 사용 흐름

1. 브라우저 접속 → **PIN 입력** (화면에 표시되는 4자리)
2. **사진 폴더 선택** (사진이 들어있는 폴더 탐색 후 ‘이 폴더로 결정’ 클릭)
3. 각자 **신랑 / 신부** 아이콘을 선택하고 하트로 사진을 클릭하면 실시간 반영
4. 우측 상단 **‘최종 확정 & 복사’** → 목록 확인 → **복사 실행**
   - 사진 폴더 옆에 `_최종셀렉_날짜` 폴더가 자동 생성되며, 이 폴더를 작가님께 보냅니다!

## (선택) 썸네일 가속 — 2000장도 훨훨
기본은 원본 이미지를 씁니다. 사진이 많거나 폰에서 느리면 한 줄만 설치:
```bash
npm install sharp
```
이후 다시 실행하면 썸네일 캐시가 켜져 훨씬 빨라집니다. (설치 안 돼도 정상 동작)
